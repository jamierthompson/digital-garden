distinct file sets shippable in parallel. The key risk in parallel coding is two agents editing one
file; mitigated by **file-disjoint ownership** + a **separate git worktree per slice** (deps
hardlinked from the pnpm store) so concurrent builds couldn't collide. The stub `ProjectScope` uses
