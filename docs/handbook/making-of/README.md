# Making-of — the handbook itself

This directory is the **making-of the handbook**: the research, drafts, debates, and skill-evaluation
runs that produced [`docs/handbook/`](../) and the `agent-team` skill. It documents **how the handbook
was written and graded**, not product work.

- `research/` — the R-notes that fed the handbook pages.
- `round-1-drafts/`, `round-2-debate/`, `synthesis.md` — the handbook's own write→debate→synthesis trail.
- `agent-team-skill/` — sources, eval design, and verbatim `eval-runs/` used to build/measure the skill.

> **Session work does NOT go here.** Notes, spikes, decision trails, and handoffs from sessions that
> change `src/`, `studio/`, or the product belong in [`docs/sessions/`](../../sessions/) (named
> `YYYY-MM-DD-<slug>.md`, or a dated folder for a multi-file trail). See
> [`docs/sessions/README.md`](../../sessions/README.md).
